MAIN CONCEPT OF THE COURSES PAGE

-- THE ANRDROID SVG WILL BE REPLACED WITH THE APPROPRIATE SVG OF THE BRANCH
-- THE COLOR OF THE COLUMN CAN BE CHANGED ACCORDINGLY
